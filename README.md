Swifty-Extensions is a collection of native Swift extensions that you needs in your most of the Projects.

## Requirements
* Swift 5.0+
* iOS 10.0+

- Bundle Extension
- Data Extension
- Date Extension
- Dictionary Extension
- Double Extension
- Int Extension
- Locale Extension
- NSNotification+Extension
- String Extension
- UserDefaults Extension
