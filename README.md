# Swifty-Extensions

Useful swift extensions for different classes like: 


* Bool
* Double
* String
* Dictionary
* Array 
* NSDate
* NSTimer
* NSURL
* NSUserDefaults
* NSObject
* NSLayoutConstraint
* UIColor
* UIAlertController
* UIApplication
* UIImage
* UIImageView
* UITextView
* UIViewController


## Installation
---
Swifty-Extensions is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "Swifty-Extensions"
```
